# how to get here

1. go to chrome://chrome-signin (or Settings > Accounts > Add Google Account)
2. type google@d11.org
3. click Sign-in options
4. click Sign in with GitHub
5. scroll down to the bottom of the page to the footer and click Docs
6. scroll down to the bottom of the page to the Still need help section and click Ask the GitHub community
7. click the search button/icon in the top right corner of the page to the left of the Sign in button
8. type "org:croslane" and press enter
9. go to this repo (webview)
10. congratulations youre here
